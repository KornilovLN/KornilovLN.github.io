Пример полного процесса

    Клонирование репозитория:

git clone https://github.com/KornilovLN/KornilovLN.github.io.git
cd KornilovLN.github.io

Создание файла index.html и config.yml:

echo "<тут пробный текст html" > index.html
echo "theme: minima" > _config.yml

Коммит и пуш изменений:

    git add index.html _config.yml
    git commit -m "Initial commit"
    git push origin main

    Настройки GitHub Pages:
        Перейдите в Settings.
        Найдите раздел Pages.
        Выберите ветку main и папку /root.
        Нажмите Save.

Сайт искать на https://KornilovLN.github.io/.
